Task 1: Student Score Prediction
The goal of this task was to predict students’ exam scores based on their study hours using a supervised machine learning approach. I worked with the Student Performance Factors dataset from Kaggle, performing data cleaning, exploration, and basic visualization to understand patterns and relationships in the data.

I split the dataset into training and testing sets and trained a linear regression model to estimate final scores. Model performance was evaluated using appropriate regression metrics, and predictions were visualized to compare with actual scores.

As a bonus, I explored polynomial regression and experimented with additional features such as sleep and participation to see how they affected model performance.

Key learnings and insights:

Importance of data cleaning and exploratory analysis before modeling

How to implement linear and polynomial regression in Python

How feature selection can impact model accuracy

Visualizing predictions helps in interpreting model performance and patterns

Tools & Libraries: Python, Pandas, Matplotlib, Scikit-learn
